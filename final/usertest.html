FINAL PROJECT USER TEST RESULTS

Test Observation

1. Users can find the clickable images and the buttons quickly and they used a relatively small amount of time to read through all the text and contents.

2. Overlay windows kept overlapping with each other and the text on the overlays stacked together.

3. Some users skip the bottom two sections once they know the page goes with the same pattern.

Reflection

With all the helpful feedback from users, I can increase the readability of the overlays. The current version is hard to read with the background behind it. Try to increase the opacity of the white background color of the overlay. For the overlapping issue of the overlay windows, I am going to work on the JS and add a function to hide the previous overlay post. To make the project more interesting and attractive, I can separate different sections with the different theme colors instead of using all blue colors. I can also add some buttons to give the user an option to go to the next post or previous post.
